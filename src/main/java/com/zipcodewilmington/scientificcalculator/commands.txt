Available commands:
    Functional:
    -toggleunits (toggle through degrees/radians)
    -switchunits (set units manually)
    -toggledisplay (toggle through binary/decimal/hexadecimal/octal)
    -switchdisplay (set display manually)
    -clear (set current value to 0.0)
    -close (exit program)
    -help (show list of commands)

    Memory:
    -memoryadd (Add the displayed value to the value stored in memory, updating both)
    -resetmemory (clear the value stored in memory)
    -reallmemory (Clear the current value and set it to the value stored in memory)

    Arithmetic:
    -add
    -subtract
    -multiply
    -divide
    -sqRoot (square root)
    -square
    -exp
    -absolutevalue
    -random (sets value to a random number)

    Trigonometry:
    -degToRad (convert degrees to radians)
    -radToDeg (convert radians to degrees)
    -sine
    -cosine
    -tangent
    -arcsine
    -arccosine
    -arctangent

    Scientific:
    -commonlog
    -inversecommonlog
    -naturallog
    -inversenaturallog
    -factorial
